# 🐍 Neon Snake Game

A modern neon-themed implementation of the classic Snake game built entirely with **HTML5 Canvas, CSS3, and Vanilla JavaScript**.

The game features smooth gameplay, responsive controls, dynamic sound effects generated using the Web Audio API, local high-score persistence, and a premium cyberpunk-inspired user interface — all contained within a single HTML file.

---

## ✨ Features

* Classic Snake gameplay
* Neon-inspired visual design
* Smooth Canvas-based rendering
* Responsive desktop and mobile experience
* Keyboard controls
* Touch button controls
* Swipe gesture support
* Dynamic sound effects using Web Audio API
* High-score saving using Local Storage
* Game over and restart system
* Lightweight and dependency-free

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* HTML5 Canvas API
* Web Audio API
* Local Storage API

---

## 🎮 Controls

### Desktop

| Key   | Action          |
| ----- | --------------- |
| W / ↑ | Move Up         |
| S / ↓ | Move Down       |
| A / ← | Move Left       |
| D / → | Move Right      |
| Space | Start / Restart |
| Enter | Start / Restart |

### Mobile

* Swipe Up → Move Up
* Swipe Down → Move Down
* Swipe Left → Move Left
* Swipe Right → Move Right

Or use the on-screen directional buttons.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/neon-snake-premium.git
```

Open the project directory and launch:

```text
index.html
```

in any modern web browser.

No installation or dependencies are required.
